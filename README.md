# OTC Pharmaceutical Sales Analysis

## Overview
This project analyzes OTC pharmaceutical sales data using SQL to identify sales trends, product performance, and country-level revenue distribution.

## Dataset
The dataset includes:
- Date
- Product
- Sales Person
- Boxes Shipped
- Revenue
- Country

## Tools Used
- MySQL
- SQL
- Excel
- Power BI

## Key Analysis
- Total revenue generated
- Product-wise revenue
- Country-wise sales distribution
- Salesperson performance
- Monthly sales trends

## Project Structure
dataset/ – raw dataset  
sql/ – SQL analysis queries  
dashboard/ – Power BI dashboard  
images/ – dashboard screenshots
